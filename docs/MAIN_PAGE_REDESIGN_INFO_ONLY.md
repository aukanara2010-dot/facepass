# Редизайн главной страницы: Информационная версия

## Обзор изменений

Главная страница (/) переделана в информационную версию без функциональных кнопок, соответствующую дизайну https://facepass.pixorasoft.ru/

## Ключевые изменения

### 1. Убраны функциональные элементы
- ❌ Кнопки "Снять с камеры" и "Загрузить фото"
- ❌ Модальные окна (камера, загрузка)
- ❌ JavaScript функциональность поиска
- ❌ Кнопка "Личный кабинет"

### 2. Добавлен информационный блок
```html
<div class="bg-blue-50 border border-blue-200 rounded-2xl p-6">
    <h3>Как получить доступ к поиску?</h3>
    <p>Для поиска ваших фотографий обратитесь к фотографу...</p>
</div>
```

### 3. Обновлен блок "Как это работает"
- **Шаг 1**: Получите ссылку от фотографа
- **Шаг 2**: Загрузите селфи (на странице сессии)
- **Шаг 3**: Получите свои фото

### 4. Брендинг PIXORASOFT
- ✅ PHOTO POINT → PIXORASOFT
- ✅ Описание: "Быстро, точно и удобно"
- ✅ Копирайт: © 2026 PIXORASOFT

### 5. Упрощенный JavaScript
Оставлена только базовая функциональность:
- FAQ аккордеон
- Плавная прокрутка по якорям
- Скрытие/показ шапки при скролле

## Структура страниц

### Главная страница (/)
- **Назначение**: Информационная страница о сервисе
- **Функциональность**: Только просмотр информации и FAQ
- **Призыв к действию**: Обратиться к фотографу за ссылкой

### Страница сессии (/session/{id})
- **Назначение**: Функциональная страница поиска
- **Функциональность**: Полный поиск по лицу
- **Призыв к действию**: Загрузить селфи и найти фото

## Пользовательский сценарий

1. **Пользователь заходит на главную** → видит информацию о сервисе
2. **Читает FAQ** → понимает как работает система
3. **Обращается к фотографу** → получает персональную ссылку
4. **Переходит по ссылке** → попадает на функциональную страницу
5. **Загружает селфи** → находит свои фотографии
